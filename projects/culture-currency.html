The challenge

Culture Currency is an upcoming streetwear fashion store. I have been tasked with designing and developing their e-commerce store.

The goal

Create a comprehensive UI kit for the brand, and then adapt it into a website. In order to design the UI/UX, I used Adobe XD. The UI kit is being used to develop a custom theme for the Wordpress CMS, on which the e-commerce store is being built.

the process

When handed the guidelines for the Culture Currency visual identity, there were many opportunities to explore digital implementations. The question was, how would we achieve a practical site design, while still being complementary to the experimental brand standards? We needed to accommodate a fluid user experience while designing an experimental website that would stand out from its competitors. These would need to be taken into account when crafting UI elements – buttons, forms, navigation styling, etc.
